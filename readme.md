# Internal Communication Twin

An AI assistant that helps employees write internal communication in a consistent professional tone.

## About

Build an AI character that helps employees draft internal emails, Slack messages, and internal wiki documentation in their own professional voice and style.

## Data Sources

- Emails & Slack Messages
- Documents (Privacy / Code of Conduct)

## Core Features 
1. Text -> Professional Message
User input:
“tell team meeting moved to 3pm”

AI output:
“Hi team, just a quick update that today’s meeting has been moved to 3 PM…” 
2. Tone Control
User selects:
Formal
Friendly
Direct
👉 Same message, different tone
Example:
Formal → “Please be informed that…”
Friendly → “Just a quick heads-up…”
3. Message Type Selection
User chooses:
Email
Slack message
Announcement
4. Rewrite / Improve Feature
User pastes:
“send me report asap”
AI rewrites:
“Hi, could you please send me the report as soon as possible? Thank you.”
5. Templates / Quick Actions
Buttons like:
“Write a reminder”
“Write an apology”
“Write a follow-up”

Web interface:
A main text box for the user's rough draft
A drop-down menu for "Platform"
A drop-down menu for "Tone"

Why this system?: This Internal Communication Twin empowers employees to write faster and more consistently by automating brand-aligned messaging , while utilizing company data in a morally sound, secure way that respects user privacy.
